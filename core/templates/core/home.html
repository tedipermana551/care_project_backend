Infrastructure

Where everything lives

🐘
Database
Neon (PostgreSQL)
Serverless Postgres that scales to zero. SSL-required connection via DATABASE_URL. Falls back to SQLite locally.
☁️
File Storage
Cloudflare R2
S3-compatible object storage for user avatars. Zero egress fees. Activated when USE_R2=True.
🤗
Deployment
Hugging Face Spaces
Dockerized and running on port 7860. Auto-migrates on every cold start via Gunicorn.

How to authenticate

JWT Bearer Tokens

After logging in, include your token on every request:

Authorization: Bearer <your_access_token>

Use the POST /api/auth/token/refresh/ endpoint with your refresh token to get a new access token when it expires.
